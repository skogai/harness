# Codex configuration

This directory is generated by Agent Starter when `--agent codex` or `--agent all` is selected.

## What's here

| Path | Purpose |
|---|---|
| `.codex/skills/<skill-id>/SKILL.md` | Project-local skill instructions generated from the shared Agent Starter skill source. |
| `AGENTS.md` | Root Codex instruction file that tells Codex when to read each local skill. |

Codex reads `AGENTS.md` as project guidance. The skill files live under `.codex/skills/` so the project can keep richer, progressive-disclosure instructions without putting every skill body into the root instruction file.

## Extending

Add new shared skills to the package template source, then reinstall with `--agent codex` or `--agent all`. Avoid editing generated `.codex/skills/*/SKILL.md` files by hand unless you want project-specific overrides.

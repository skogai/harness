# Copywriting Frameworks

## Overview

Use this skill to turn generic messaging into specific persuasive copy. Anchor the work on one reader, one desired action, believable proof, and the next sentence.

## Fast Workflow

1. Identify the audience, offer, desired action, channel, funnel stage, brand voice, proof constraints, and forbidden claims. If details are missing and the task can still proceed, state assumptions briefly.
2. Choose the dominant job:
   - Hook/headline: select one reader, open a curiosity loop, and use a concrete image or identity target.
   - Long-form copy: use AIDA and story; make the customer the hero and the product the guide.
   - Ad or video script: start with a pattern interrupt, build belief before naming the product, then make the CTA feel earned.
   - Critique: diagnose attention, specificity, proof, desire, objections, and CTA friction.
   - Copywork: build a repeatable swipe-study routine from strong examples in the same medium.
3. Draft in passes: first make it specific, then make it credible, then cut.
4. Mark unverifiable claims as `needs proof` instead of inventing data, testimonials, endorsements, or scientific support.

## Required Checks

- Make the first line earn the second line.
- Make the headline select or repel a specific audience.
- Add proof, a concrete demonstration, or an explicit proof placeholder.
- Create tension and change; do not merely list events.
- Handle likely objections before the CTA.
- Keep sentences simple and varied; prefer one point per sentence.
- Let copy be long only while tension, proof, or desire is still building.
- Match the brand's permission for humor, edge, and directness.

## Output Shape

For creative requests, provide several variants grouped by angle. For critique requests, lead with the highest-leverage fixes, then give a revised version. For strategic requests, show the selected framework and the reason it fits the audience, channel, and funnel stage.
